인간의 신경망과 뉴런

인간의 두뇌는 뉴런이 시냅스를 통해 병렬적으로 연결됩니다.

  • 각 뉴런은 여러 입력 신호의 합을 받습니다
  • 합이 역치(threshold)보다 크면 발화(fire)하여 다음 뉴런에 신호를 전달합니다
  • 시냅스를 따라 전달된 신호가 다음 뉴런의 역치를 넘으면 연쇄 반응이 일어납니다
대기 신호 수신 발화 (역치 초과)

시냅스를 따라 이동하는 자극 — 역치를 넘으면 뉴런이 점등됩니다

퍼셉트론 — 인공 뉴런

퍼셉트론은 인공신경망의 기본 단위로, 생물학적 뉴런을 수학적으로 모델링합니다.

p(x, y) = 1.0x + 1.0y + −1.5
활성화: p(x, y) > 0 → T / F
p > 0 (활성) p ≤ 0 (비활성) 결정 경계 (점멸)

점을 클릭하여 입력을 추가 · 슬라이더로 결정 경계(직선)를 조절하세요

단층 퍼셉트론의 한계

하나의 직선(결정 경계)으로 입력을 분류할 수 있는지 확인합니다.

AND: 두 입력이 모두 1일 때만 출력 1 — 선형 분리 가능 ✓

x₁x₂출력

✓ 단일 직선으로 완벽 분류 가능!

AND: 하나의 직선으로 0/1을 분리할 수 있습니다

다층 퍼셉트론 (MLP)

XOR 문제는 단층 퍼셉트론으로 해결할 수 없지만, 은닉층을 추가하면 해결됩니다.

입력 x₁ x₂
→
은닉층 NAND OR
→
출력 AND

출력 = AND( OR(x₁,x₂), NAND(x₁,x₂) ) = XOR(x₁, x₂)

x₁x₂ORNANDXOR

단층 퍼셉트론 — XOR 불가

하나의 직선으로는 XOR 분류 불가

다층 퍼셉트론 — XOR 해결

두 개의 경계로 XOR 영역 분리
