# 통합 내용

- `school-main`을 기본 프로젝트로 유지했습니다.
- `neural-vision-lab`의 전체 파일을 `page3/`에 추가했습니다.
- 루트 `index.html`에 `Neural Vision Lab` 탭을 추가했습니다.
- 기존 `page1`과 `page2`의 파일 및 경로는 변경하지 않았습니다.

## 실행

정적 서버에서 `school-main/index.html`을 여세요.

예시:

```bash
cd school-main
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 접속하면 됩니다.
